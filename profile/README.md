## 추진 배경
- 농구에 관심이 많은 사람들이 현재 농구 인프라에 대한 불편함을 느껴 그동안의 경험을 바탕으로 불편사항을 해소하고자 앱 개발 기획
  
## 프로젝트 목표
- 농구 애호가들이 모든 농구 활동(픽업 게임, 동호회, 리그 & 토너먼트)을 간편하고 편리하게 즐길 수 있도록 지원하는 커뮤니티 플랫폼 개발
